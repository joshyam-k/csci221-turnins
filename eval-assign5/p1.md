```
addi $t0, $zero, 2  # $t0 = 2 we will compare this to n to know when to stop
add $t1, $zero, $zero  # $t1 = 0 this will hold f_(n-1)
addi $t2, $zero, 1     # $t2 = 1 this will hold f_(n-2)
Loop:
    sll $s0, $a0, $t0        # (checking if n < i)
    bne $s0, $zero, Exit     # if $s0 is one then i <= n is no longer true so we exit
    add $t3, $t1, $t2        # f_n = f_(n-1) + f(n-2)
    addi $t2, $t1, 0         # f_(n-2) (new) = f(n-1) (old)
    addi $t1, $t3, 0         # f_(n-1) (new) = f(n) (old)
    addi $t0, $t0, 1         # $t0 = $t0 + 1 (i++)
Exit:
    add $v0, $t3, $zero
```
